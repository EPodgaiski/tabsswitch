# jQuery tabsSwitch

Небольшой скрипт для анимированного переключения табов на странице.

## Подключение:
 - Подключаем библиотеку jQuery 1.10.0+
 - Добавляем фунцию из файла [js/tabs.switch.js](js/tabs.switch.js)
 - Проставляем классы в HTML:
  _.js_tabs_ - область действия скрипта
  _.js_tab_btn_ - кнопка переключения таба
  _.js_tabs_content_ - область с контентом для каждой кнопки
 - Прописываем атрибут для каждой кнопки:
  _data-target-id="target_item_ID"_, где 'target_item_ID' - это ID блока с контентом для этой кнопки
 - Запускаем скрипт на странице:
```js
 $('document').ready(function(){
    tabsSwitch();
  });
```
 - Прописываем стили для табов (есть в примере в [css/styles.css](css/styles.css)). (стили выделены /*THIS CSS*/)

## Параметры:
 **data-target-id="target_item_ID"** - 'target_item_ID' - это ID блока с контентом для этой кнопки.
 
 **tbsDuration** - переменная в скрипте, отвечает за скорость анимации в ms.
